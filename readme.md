## Install

### Notices
1. DocumentRoot /public
2. Required PHP >= 5.5.9

### Advanced

Install laravel vendors
```
$ Composer install
```

Install front-end assets

```
$ bower install
```

Setup config    
Edit DATABASE config

```
$ vim .env
```

Install database

```
$ php artisan migrate --seed
```
