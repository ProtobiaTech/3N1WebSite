--- layout: docs title: Docs permalink: /docs/install/ --- ## 简易安装 ### 获取程序 在[https://github.com/dev4living/3N1WebSite/releases](https://github.com/dev4living/3N1WebSite/releases) 下载安装程序 ### 安装 1. 把下载的3N1WebSite解压存放在WEB服务器中，并分配域名 2. 访问设定好的域名，进行安装。需要注意的是目录权限(根目录 和 /bootstrap/cache目录 和 /.env文件 需要WEB服务器可写权限) 3. 跟随安装向导进行，即可完成安装 ### 其他 遇到任何问题请联系我们，我们会给序强有力的支持 Giter: [https://gitter.im/dev4living/3N1WebSite](https://gitter.im/dev4living/3N1WebSite) QQ群: 242078519
